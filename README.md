# panda-ui
> awesome mobile ui lib implement by Vue


## Install

```shell
npm i -S panda-ui
```
## Usage

```
import Vue from 'vue'
import panda from 'panda'
Vue.use(panda)
```

## Use modularized panda-ui

```
import {Style, VButton} from 'panda/src/module'
```

## Development


```shell
git clone
cd panda-ui
npm install
npm start
```

## Build

